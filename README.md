# рҹ“қ Smart Notes

A modern, responsive, and lightweight note-taking web application built using **HTML**, **CSS**, and **JavaScript**.

Smart Notes helps users quickly create, organize, search, and manage notes through an intuitive interface. All notes are automatically saved in the browser using **Local Storage**, ensuring data persists even after refreshing or reopening the application.

---

##  Features

-  Create unlimited notes
   Search notes instantly
-  Random note colors for better organization
-  Dark & Light mode support
-  Automatic Local Storage
-  Export all notes as a `.txt` file
-  Delete notes easily
-  Responsive design for desktop and mobile devices
-  Smooth animations and modern UI

##  How It Works

- Click **Create Notes** to add a new note.
- Type directly inside the note card.
- Search any note using the search bar.
- Toggle between **Light** and **Dark** mode.
- Export all notes into a text file.
- Notes are automatically saved using Local Storage.
